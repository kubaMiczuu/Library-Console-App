# 📚 Library Management System (CLI)

## 🇵🇱 Opis projektu

Konsolowa aplikacja do zarządzania zbiorami bibliotecznymi.  
Projekt skupia się na bezpieczeństwie danych, testowalności oraz przejrzystej separacji warstw aplikacji.

---

## ✨ Kluczowe funkcjonalności

- 📖 Zarządzanie książkami (**CRUD**)
- 💾 Persystencja danych w formacie **JSON** *(Jackson)*
- 🧱 Architektura warstwowa:
  - Model
  - Service
  - Repository
- 🔒 Niemutowalne obiekty:
  - `@Builder`
  - `@Jacksonized`

---

## 🛠️ Technologie

- **Java 17+**
- **Maven**
- **Lombok**
- **Jackson**
- **JUnit 5**
- **Mockito**
- **AssertJ**

# 🇬🇧 Project Overview

A console-based application for library management.  
The project focuses on data integrity, testability, and clean separation of concerns.

---

## ✨ Key Features

- 📚 Book management (**CRUD**)
- 💾 File-based data persistence in **JSON** format *(Jackson)*
- 🧱 Layered architecture:
  - Model
  - Service
  - Repository
- 🔒 Fully immutable objects:
  - `@Builder`
  - `@Jacksonized`

---

## 🛠️ Tech Stack

- **Java 17+**
- **Maven**
- **Lombok**
- **Jackson**
- **JUnit 5**
- **Mockito**
- **AssertJ**
